class ShellTask

package vscode.proposed

A task that executes a shell command.

Constructor

new (name:String, commandLine:String, ?problemMatchers:EitherType<String, Array<String>>)

new (name:String, commandLine:String, options:ShellTaskOptions, ?problemMatchers:EitherType<String, Array<String>>)

Creates a shell task.

Parameters:

name

the task's name. Is presented in the user interface.

commandLine

the command line to execute.

options

additional options used when creating the shell.

problemMatchers

the problem matchers to use, like '$tsc' or '$eslint'. Problem matchers can be contributed by an extension using the problemMatchers extension point.

Variables

read onlycommandLine:String

The command line to execute.

group:Null<String>

The task group this tasks belongs to. See TaskGroup for a predefined set of available groups. Defaults to undefined meaning that the task doesn't belong to any special group.

identifier:Null<String>

The task's identifier. If omitted the internal identifier will be ${extensionName}:${name}

isBackground:Bool

Whether the task is a background task or not.

read onlyname:String

The task's name

options:ShellTaskOptions

The shell options used when the shell is executed. Defaults to an empty object literal.

problemMatchers:Array<String>

The problem matchers attached to the task. Defaults to an empty array.

source:Null<String>

A human-readable string describing the source of this shell task, e.g. 'gulp' or 'npm'.

terminalBehavior:TaskTerminalBehavior

The terminal options. Defaults to an empty object literal.
