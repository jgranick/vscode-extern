class ProcessTask

package vscode

A task that starts an external process.

Constructor

new (name:String, process:String, ?problemMatchers:ProblemMatchers)

new (name:String, process:String, args:Array<String>, ?problemMatchers:ProblemMatchers)

new (name:String, process:String, args:Array<String>, options:ProcessOptions, ?problemMatchers:ProblemMatchers)

Creates a process task.

Parameters:

name

the task's name. Is presented in the user interface.

process

the process to start.

problemMatchers

the problem matchers to use.

Variables

args:Array<String>

The arguments passed to the process. Defaults to an empty array.

group:TaskGroup

The task group this tasks belongs to. Defaults to undefined meaning that the task doesn't belong to any special group.

identifier:String

The task's identifier. If omitted the name is used as an identifier.

isBackground:Bool

Whether the task is a background task or not.

read onlyname:String

The task's name

options:ProcessOptions

The process options used when the process is executed. Defaults to an empty object literal.

problemMatchers:Array<String>

The problem matchers attached to the task. Defaults to an empty array.

read onlyprocess:String

The process to be executed.

terminal:TerminalBehaviour

The terminal options. Defaults to an empty object literal.
