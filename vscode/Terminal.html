typedef Terminal

package vscode

An individual terminal instance within the integrated terminal.

Properties

read onlyname:String

The name of the terminal.

@readonly

dispose ():Void

Dispose and free associated resources.

hide ():Void

Hide the terminal panel if this terminal is currently showing.

sendText (text:String, ?addNewLine:Bool):Void

Send text to the terminal. The text is written to the stdin of the underlying pty process (shell) of the terminal. Note that this will currently force the terminal panel to the foreground, this is changing in v1.6 such that it will require an explicit call to Terminal.show in order to show the terminal panel.

Parameters:

text

The text to send.

addNewLine

Whether to add a new line to the text being sent, this is normally required to run a command in the terminal. The character(s) added are \n or \r\n depending on the platform. This defaults to true.

show (?preserveFocus:Bool):Void

Show the terminal panel and reveal this terminal in the UI.

Parameters:

preserveFocus

When true the terminal will not take focus.
